# Mapa global del curso

Cloud se ubica en el Semestre 5 como complemento directo de Arquitectura de
software. Arquitectura enseña límites internos; Cloud enseña plataformas,
responsabilidades delegadas y costos operativos.

```mermaid
flowchart LR
    networking["Redes"] --> cloud["Cloud"]
    os["Sistemas operativos"] --> cloud
    db["Bases de datos internals"] --> cloud
    distributed["Sistemas distribuidos"] --> cloud
    architecture["Arquitectura de software"] --> cloud
    cloud --> ai["AI Engineering"]
    cloud --> travel["Travel Tech"]
    cloud --> devops["DevOps"]
```
